## Installation

```bash
$ npm i
$ npm i -g @nestjs/cli
```

## Running the app

Firstly you need to create .env from .env.example fulfilled with your options

```bash
# Development

# Run main app
$ nest start --watch

# Run tg service
$ nest start tgbot --watch


# Build for production

# Build main app
$ nest build

# Build tg service
$ nest build tgbot
```
